# Backend

Этот каталог содержит Laravel-ориентированный каркас для API портала ЦЦОП:

- `app/` - контроллеры и служебные классы.
- `routes/api.php` - карта API-маршрутов.
- `database/migrations` - миграции под PostgreSQL.
- `database/seeders` - начальное наполнение ролями и страницами.
- `public/index.php` - точка входа HTTP для контейнера разработки.

Для реального развёртывания можно развить этот каркас до полноценного приложения Laravel, сохранив уже подготовленные сущности, роли и базовую структуру API.
